# 米坛社区注册教程

:::tip
使用QQ邮箱注册需要检查是否开通QQ邮箱提醒，否则无法接收邮件。  
开启邮箱提醒 操作步骤：  
打开QQ-右滑-设置-通用-辅助功能-开通QQ邮箱提醒
:::

:::tip
建议使用带有Chrome内核的浏览器，以获得更佳的浏览体验。
:::

使用浏览器，打开[米坛社区(bandbbs.cn)](https://bandbbs.cn/),点击上方导航栏的“立即登录",或者点击下方链接直达注册页面:

| [注册米坛社区Bandbbs账号](https://www.bandbbs.cn/login/register)|
|---------| 

这里以使用**QQ邮箱**注册做示例。

1. 进入注册账号页面，输入你的邮箱进行注册（QQ邮箱格式为你的QQ号加上@qq.com）
![](<resources/md_images_out/折腾米环、米表完整图文教程/米坛社区注册教程/01_2 此时会进入米坛社区的登录页面，点击_登录按钮_下方的_立即注册_进入注册账号页面，输入你的邮箱进行注册（建议使用QQ/0005_3ed4bf0691.png>)

2. 点击"发送验证码到邮箱"之后会有一条验证消息发到你的邮箱（[mail.qq.com](https://mail.qq.com)）里，就像下面这样，我们点击"确认您的邮件"
![](<resources/md_images_out/折腾米环、米表完整图文教程/米坛社区注册教程/02_3 点击_发送验证码到邮箱_之后会有一条验证消息发到你的邮箱（[mail.qq.com](https___mail.qq/0006_29ed5d8822.png>)

然后开始验证，此时验证码会自己输入"米坛社区注册验证码"的文本框中，然后依次输入你的用户名和密码，注册完成
![](<resources/md_images_out/折腾米环、米表完整图文教程/米坛社区注册教程/02_3 点击_发送验证码到邮箱_之后会有一条验证消息发到你的邮箱（[mail.qq.com](https___mail.qq/0007_03698ef418.png>)
