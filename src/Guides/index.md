# 快速开始

在开始折腾前，请你了解下面内容：

1. 你需要 拥有一个米坛社区Bandbbs账号
2. 你需要有一定的动手能力 
3. 遇到问题 会正确提问问题
4. 你非常清楚：你在干什么、你需要什么

如果你不知道如何注册，请阅读：
| [米坛社区注册教程](/Guides/米坛社区注册教程.html) |
|---------|
