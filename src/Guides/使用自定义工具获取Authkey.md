# 使用自定义工具获取Authkey

## 提取运动健康log

1. 打开`小米运动健康`，正常连接设备，点击同步。

2. 在下方选择`我的`

![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0044_ccc823af8d.jpg)

3. 在“我的”界面下滑找到关于
![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0045_dc4fb49cca.jpg)

4. 在关于界面，点击logo 10次
![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0046_a291a8d7a4.jpg)

5. 点击确定
![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0047_05f934ec09.jpg)

## 读取Authkey

1. 返回自定义工具，在下面找到`工具`

![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0048_87f938bec4.jpg)

2. 然后下滑界面 选择“AuthKey读取工具”
![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0049_1c64b847ce.jpg)

3. 划到下面，点击读取
![](./Res/md_images_out/折腾米环、米表完整图文教程/快速了解表盘自定义工具使用/07_分类-动态表盘/0050_95d793b1c8.jpg)

把请求的权限全部同意就好了

下面有显示设备以及AuthKey就是成功了，不成功的话检查下是不是权限没有给全
