## 🌟 Q1：米坛账号无法登录？

首先请你检查网络状况，并多次尝试。建议首选使用米坛社区官方客户端进行登录，其次将你的默认浏览器设置为 Chrome 或 Microsoft Edge，别用系统自带浏览器，夸克和QQ浏览器之类的也最好别用

## 🌟 Q2：小米账号无法登录？

你可以使用<mark>手机号 或 米ID 或 邮箱＋密码</mark>登录

如果外区无法使用手机号登录，请等待后续版本更新

如果登录一直转圈：可以尝试检查账号下是否有穿戴设备、更改密码等操作，如果依旧不行请带 log 与我们反馈

### 关于小米账号登录

AstroBox 部分功能涉及通过调用小米账号服务接口以访问资源或启用特定设备操作。在此过程中，用户提供的账号凭证信息（包括用户名、密码或授权票据等）将直接提交至小米官方服务器，以完成身份验证。

本软件不会在未经用户同意的情况下存储、记录或篡改用户凭证信息。然而：

- 该传输行为的安全性完全依赖于小米服务器及其接口实现机制，非本软件所能控制或担保

- 若因接口变化、服务策略调整或安全机制触发而导致用户账号受到限制、登录失败或产生数据问题，本软件及其开发者不承担相关责任；

- 用户应审慎判断是否使用此类功能，并对涉及账号凭证的使用风险承担全部责任。

我们强烈建议用户仅在可信网络环境中操作，并在授权登录前评估其账号安全性要求。
